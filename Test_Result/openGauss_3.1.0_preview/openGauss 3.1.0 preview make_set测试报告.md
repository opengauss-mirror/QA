# 1     特性概述

openGauss在兼容B库情形下安装dolphin插件，实现兼容MySQL的make_set函数。make_set函数首先会将bits位的值转换为二进制，然后将二进制翻转，利用翻转后的每一位二进制与其相对应的str值按位与，如得到的值为零或null的情况下会跳过，其他情况会输出。
