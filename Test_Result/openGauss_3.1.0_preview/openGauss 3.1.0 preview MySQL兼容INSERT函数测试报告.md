# 1     特性概述

openGauss在兼容B库情形下安装dolphin插件，实现兼容MySQL的INSERT(str,pos,len,newstr)函数。函数返回字符串str，开头在pos位置的子串，并且用len个字符长的字符串代替newstr。如果pos不是字符串的长度范围内，则返回原始字符串。如果len不是字符串的其余部分的长度范围内，从位置pos替换字符串的其余部分。如果参数为NULL返回NULL。
