# 1 概述

本特性设计旨在介绍如何实现以‘f/F’或‘d/D’结尾的浮点数的兼容性支持，以及如何支持特殊的‘NaN’和‘infinity’表示浮点数。这些功能的实现将使系统能够处理更广泛的浮点数表示方式，提高灵活性。
