# 1 概述

在opengauss数据库中，支持gms_stats高级包GATHER_SCHEMA_STATS用于收集指定schema中所有对象的统计信息。

使用方式如下：

--创建插件
**create extension gms_stats;**
--调用函数进行收集
**gms_stats.gather_schema_stats('schema_name');**
--查询收集到的信息

**select * from pg_stats where schemaname = 'schema_name';**
