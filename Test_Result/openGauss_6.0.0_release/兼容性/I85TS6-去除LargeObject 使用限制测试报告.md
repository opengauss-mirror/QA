# 1 概述

1. 解除opengauss中大对象使用限制
2. 从PG移植lo_from_bytea、lo_put、lo_get、lo_truncate64、lo_lseek64、lo_tell64函数和libpq接口，添加对应的测试用例，让opengauss可以操作大于2g的文件

3. 修复openguass不能import大于1.5G文件的问题

4. 修复openguass导入大于4G文件data列数据被截断问题

5. 减少开销，每个大对象描述符最多进行一次读/写权限检查，而不是每次lo_read或lo_write调用一次权限检查。
