# 1 概述

在opengauss数据库中，可通过alter trigger方式启用禁用单个触发器。

语法如下：

--ALTER TRIGGER 方式禁用触发器：

**ALTER TRIGGER [trigger_name] DISABLE;**

--ALTER TRIGGER 方式启用触发器：

**ALTER TRIGGER [trigger_name] ENABLE;**
