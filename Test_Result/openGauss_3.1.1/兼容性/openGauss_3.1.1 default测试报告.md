# 1     特性概述

openGauss在兼容B库情形下安装dolphin插件，实现兼容MySQL的default函数。MySQL中，默认值的引用通过default函数做到，这部分在使用和处理上，更类似语法而非实际函数，其相当于运行中会替换为默认值的表达式。如果没有默认值，或者默认值为一个函数（时间类型），那么返回NULL。
