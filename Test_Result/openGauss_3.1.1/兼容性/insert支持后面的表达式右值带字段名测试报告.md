# 1 特性概述

openGauss在兼容B库情形下，安装dolphin插件，可以实现对MySQL中insert语法表达式右值支持带字段名的兼容。该语法按 SQL 语句顺序来执行赋值表达式，当遇到字段引用时，返回该字段的输出值作为引用值，若被引用字段的默认表达式为常量则将输出值的初始值设置为默认值，否则将输出值的初始值设置为null。
