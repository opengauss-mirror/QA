# 1     特性概述

openGauss在兼容B库情形下安装dolphin插件，实现兼容M语法 show status 的功能，此语法可输出数据库运行时节点统计、内存占用、事务负载、IO统计等信息，用于数据库诊断、监控等。

- 语法介绍：
```sql
SHOW [GLOBAL | SESSION] STATUS
    [LIKE 'pattern' | WHERE expr]
```
