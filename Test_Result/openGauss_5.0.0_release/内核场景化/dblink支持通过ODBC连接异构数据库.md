# 1     特性概述

dblink支持i通过加载异构数据库的ODBC链接远程数据库，实现对远程数据库的链接、SQL操作、断连。

## 3.1   测试整体结论

共设计测试用例37个功能用例，主要包含dblink通过libpq连接远程openGauss数据库、通过ODBC连接远程openGauss数据库、通过ODBC连接异构数据库，及dblink常用函数功能的验证。共计发现4个问题，均已修复且回归验证通过，整体质量良好。
