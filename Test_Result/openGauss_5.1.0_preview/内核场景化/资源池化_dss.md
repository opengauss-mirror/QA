# 1     特性概述

openGauss实现资源池化DSS黑匣子，黑匣子日志用于记录dssserver崩溃时进程、线程的堆、栈内存及寄存器等现场信息。通过黑匣子日志，可以分析出故障发生时的进程上下文，方便定位故障原因。。

openGauss实现对DSS的线程池优化，减少DSS的资源占用。
