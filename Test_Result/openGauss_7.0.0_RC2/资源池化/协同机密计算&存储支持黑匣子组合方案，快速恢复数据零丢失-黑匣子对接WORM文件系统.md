# 1 概述

本报告主要测试WR工具对接worm文件系统，以及在WR单节点模式下，通过本地连接、远程连接等方式验证相关API的基础功能、可靠稳定性，以及关键性能满足指标要求，涉及特性如下：

* 协同机密计算&存储支持黑匣子组合方案，快速恢复数据零丢失-黑匣子对接WORM文件系统
