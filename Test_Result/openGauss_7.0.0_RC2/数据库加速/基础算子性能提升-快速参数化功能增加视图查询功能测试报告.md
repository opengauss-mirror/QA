# 1 概述

该测试需求的目的是对于7.0.0RC1中，快速参数化需求功能的完善，RC1中该功能无法直观看到缓存的记录，本次需求快速参数化支持视图查询，能查询快速参数化列表里的内容，返回当前会话自身保存快速参数化的详情，提升了易用性。
