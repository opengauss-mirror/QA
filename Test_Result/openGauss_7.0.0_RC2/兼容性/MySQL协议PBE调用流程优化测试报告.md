# 1 概述

openGauss当前对于MySQL的PBE协议，是通过构造`prepare stmt from xx`、`execute stmt using xx`的做法执行对应的statment，从而实现类似PBE的效果。这样每次在做execute时都需要完整走一变parser流程，存在一定性能损耗。
