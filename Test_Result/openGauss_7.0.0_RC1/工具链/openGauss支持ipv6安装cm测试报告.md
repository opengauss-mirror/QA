## 3.1 测试结论总结
本次测试涉及OM、CM工具用例自动化连跑及cm数据库安装、升级、卸载、扩容、缩容方面，新增测试用例41个，yat连跑涉及18个测试套，所有被测特性的测试用例执行覆盖率达到100%，开发质量良好
## 3.2 约束说明

1.ipv4和ipv6不能混用 <br/>2.环境为ipv6配置<br/>3.使用普通用户安装<br/>
