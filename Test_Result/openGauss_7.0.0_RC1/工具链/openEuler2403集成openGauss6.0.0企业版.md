# 1 概述

本次特性是将之前集成的openGauss 5.0.1轻量版替换为6.0.0企业版，完整支持openGauss内核功能、dolphin插件功能。约束及限制：不包含mot功能、om、cm等外部组件，只有纯数据库内核功能。
