###  4.2.2 可靠性测试结论

| 测试步骤                                                     | 测试结果                                                   |
| ------------------------------------------------------------ | ---------------------------------------------------------- |
| 1、stop备机，主机创建段页式压缩表，备机执行全量build         | 备机重建成功，主备数据一致                                 |
| 2、stop备机，主机创建段页式压缩表并插入数据，备机执行全量build | 备机重建失败，集群不能恢复（IBOFHT，本需求不支持备份恢复） |
| 3、主机创建段页式压缩表后插入数据，主备进行switchover        | switchover失败，集群可恢复                                 |
| 4、主机创建段页式压缩表后，主备切换后，在新主上插入数据，查询主备数据是否一致 | 主备切换成功，但主备数据不一致（IBOK5O）                   |
| 5、主机创建段页式压缩表后插入数据，stop主机，备机进行failover | failover失败，不插入数据可以成功                           |
