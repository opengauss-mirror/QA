# 1 概述

本报告主要测试：1、节点变更的情况下，cm、dss、gaussdb进程退出逻辑优化，保证各服务可以优雅推出；2、异常场景下openGauss io操作优化，因为DSS故障导致的文件操作失败，应避免直接PANIC导致coredump，并增加充实或逃生功能。
