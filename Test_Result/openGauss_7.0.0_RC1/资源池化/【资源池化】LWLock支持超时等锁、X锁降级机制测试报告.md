# 1 概述

该测试需求的目的是对资源池化场景中LWLock进行优化，资源池化方案中，需要使用到LWLock，但是又必须具有超时机制，目前内核侧LWLock并不具有超时等待的能力。本需求通过增强LWLock能力来进行优化，实现LWLock超时等锁、LWLock的X锁降级等机制，优化部分场景性能低下的情况。
